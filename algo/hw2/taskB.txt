Дан список из N элементов, которые принимают целые значения от 0 до 100. Отсортируйте этот список в порядке неубывания элементов. Выведите полученный список.

Входные данные

На одной строке дан массив из N элементов. (1 ≤ N ≤ 2·10^5) — количество элементов в массиве.
Гарантируется, что все элементы массива принимают целые значения от 0 до 100.

Выходные данные

Выведите отсортированный список элeментов

Пример
Входные данные
7 3 4 2 5

Выходные данные
2 3 4 5 7 

Примечание: Использовать встроенные функции сортировки нельзя.
