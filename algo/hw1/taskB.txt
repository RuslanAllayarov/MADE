Дан массив целых чисел. Ваша задача — отсортировать его в порядке неубывания.

Входные данные
В первой строке входного файла содержится число N (1 ≤ N ≤ 100 000) — количество элементов в массиве.
Во второй строке находятся N целых чисел, по модулю не превосходящих 10^9.

Выходные данные
В выходной файл надо вывести этот же массив в порядке неубывания, между любыми двумя числами должен стоять ровно один пробел.

Пример
Входные данные
10
1 8 2 1 4 7 3 2 3 6

Выходные данные
1 1 2 2 3 3 4 6 7 8 

Примечание: Запрещается использовать стандартные сортировки.
